## <a name="P-4">Παραδοτέο 4:</a>
### **Απαντήσεις στο Quiz 4**  
**Γιατί υπάρχει αντίσταση απέναντι στις νέες γλώσσες προγραμματισμού υψηλού επιπέδου; Να δώσετε ένα παράδειγμα από την δική σας εμπειρία.**  
Οι νέες γλώσσες προγραμματισμού υψηλού επιπέδου δεν μπορούν να προσαρμοστούν στα δεδομένα που ισχύουν για τις παλαιότερες γλώσσες. Οι χρήστες μπαίνουν στη διαδικασία να αλλάξουν τον τρόπο που έχουν μάθει να προγραμματίζουν καθώς η αντίσταση από τις νέες γλώσσες προγραμματισμού είναι αντιληπτή. Από τη δική μου εμπειρία μπορώ να αναφέρω ένα παράδειγμα γλώσσας προγραμματισμού υψηλού επιπέδου, τη C, η οποία διαφέρει από τις προηγούμενες γλώσσες στον τρόπο γραφής.

**Προγραμματίζουμε την ιστοσελίδα της σχολής χωρίς στυλ και ετικέτες! Πέτυχε την πρόβλεψη του ή εμείς κάνουμε κάτι διαφορετικό από τον συνηθισμένο προγραμματισμό ιστοσελίδας;**  
Η προσέγγιση είναι τελείως διαφορετική σε σχέση με την πραγματικότητα, διότι η χρήση των ετικετών και των ετικετών και των ιστοσελίδων είναι απαραίτητη. Πιο συγκεκριμένα, δεν επιθυμούμε απλά τον προγραμματισμό σε HTML αλλά χρησιμοποιούμε CSS.

**Προγραμματίζουμε την ιστοσελίδα της σχολής με κείμενο. Γιατί πέφτει τόσο έξω στην πρόβλεψη του;**  
Είναι λογικό να πέφτει έξω στην πρόβλεψη, διότι ο προγραμματισμός μιας ιστοσελίδας μόνο με κείμενο προβλέπει ένα στυλ που δεν είναι ιδανικό, καθώς δεν εμφανίζονται ετικέτες και ιστοσελίδες.

**Ποιος είναι υπεύθυνος για το προγραμματιστικό δόγμα των ημερών μας και πως θα μπορούσαμε να σκεφτούμε με νέους τρόπους;**  
Υπεύθυνοι για το προγραμματιστικό δόγμα των ημερών είναι οι προγραμματιστές που έχουν διδάξει τον τρόπο και τις μεθόδους στις προηγούμενες γενιές.
